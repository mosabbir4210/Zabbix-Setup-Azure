# Zabbix Monitoring Setup on Azure (Beginner Friendly)

This project shows how to set up **Zabbix Monitoring on Microsoft Azure**
using **Ubuntu Server, Nginx, and PHP-FPM**.

This guide is written for **absolute beginners**.

---

## Project Goal

- Deploy Zabbix on Azure VM
- Access monitoring via browser
- Fix real PHP warning safely
- Document everything clearly

---

## Platform Used

- Cloud: Microsoft Azure
- OS: Ubuntu Server
- Web Server: Nginx
- PHP: PHP-FPM 8.4
- Monitoring Tool: Zabbix

---

## What Is Zabbix?

Zabbix is a monitoring tool that checks:
- CPU usage
- RAM usage
- Disk space
- Server health

---

## Common Issue Solved

During setup, a PHP deprecated warning appeared.
This project shows how to fix it the **right way**.
